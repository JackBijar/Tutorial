Start Server :

	1)	npm install	<= To install dependency
	
	2)	npm start	<= Start Server
	
	3)	Go to http://localhost:9966/#init	<= To generate Your ID peer Id
	
	4) Run Application into another machine and enter peer Id of first machine into Other Id textarea.
	
	5) To generate Your ID peer Id of Second machine. This ID peer Id used in first machine Other Id textarea.
	
	6) Then Click on Connect
